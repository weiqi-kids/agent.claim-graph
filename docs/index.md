用公開的課程標準與考試門檻，幫助你評估孩子的學習程度。

---

## 怎麼使用這些資料？

1. **確認目標**：孩子未來要考什麼？（學測？SAT？競賽？）
2. **查看門檻**：目標考試的分數門檻是多少？
3. **了解題型**：目標考試著重哪種思考能力？
4. **對照標準**：孩子目前年級該掌握哪些技能？

這些資料讓你用**具體數字**評估孩子的程度，而不是憑感覺。
