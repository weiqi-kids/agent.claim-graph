## 怎麼使用這些資料？

1. **確認年齡帶**：孩子幾歲？找到對應的年齡帶
2. **查看課綱**：該年齡帶各國教什麼？掌握到哪裡？
3. **了解門檻**：目標考試或競賽的分數門檻是多少？
4. **跨國比較**：不同國家的教學重點和深度有何差異？
