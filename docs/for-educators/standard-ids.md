# 標準 ID 索引

美國 Common Core State Standards (CCSS) 數學標準 ID 索引。

---

## 標準 ID 格式說明

CCSS 數學標準 ID 採用以下格式：

```
CCSS.Math.Content.{Grade}.{Domain}.{Cluster}.{Standard}
```

| 元素 | 說明 | 範例 |
|------|------|------|
| Grade | 年級 | 4, 8, HSA (高中代數) |
| Domain | 領域 | NBT, NF, OA, G, F |
| Cluster | 群組 | A, B, C |
| Standard | 具體標準 | 1, 2, 3... |
