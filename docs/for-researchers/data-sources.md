## 引用格式

如需引用本系統的資料，建議格式：

### APA 格式
```
學生學習地圖系統. (2026). {頁面標題}.
取自 https://learn.weiqi.kids/{路徑}
```

### Chicago 格式
```
「{頁面標題}」, 學生學習地圖系統,
存取於 {日期}, https://learn.weiqi.kids/{路徑}
```
