# PSLE 門檻數據

> Primary School Leaving Examination（新加坡小學離校考試）

---

## 基本資訊

| 項目 | 內容 |
|------|------|
| 考試名稱 | PSLE（Primary School Leaving Examination） |
| 主辦單位 | SEAB（Singapore Examinations and Assessment Board）/ MOE Singapore |
| 考試對象 | 小學六年級學生（約 12 歲） |
| 考試科目 | 英語、母語、數學、科學（共 4 科） |
| 成績制度 | AL（Achievement Level）1-8 |
| 總分範圍 | 4（最佳）至 32（最低） |
| 資料年度 | 2023-2024 |

---

## 成就等級（AL）制度

2021 年起，PSLE 由 T-score 制改為 AL（Achievement Level）制度。每科以 8 個等級評定：

| 等級 | 分數範圍 | 說明 |
|------|----------|------|
| AL 1 | 90 分以上 | 最頂尖表現 |
| AL 2 | 85-89 | |
| AL 3 | 80-84 | |
| AL 4 | 75-79 | |
| AL 5 | 65-74 | |
| AL 6 | 45-64 | |
| AL 7 | 20-44 | |
| AL 8 | 20 分以下 | |

> **注意**：AL 分數越低越好。4 科總分最佳為 4（每科均 AL 1），最差為 32（每科均 AL 8）。AL 1-4 的級距較窄（5 分），因為高分段的學生分布較集中。

---

## 升學分流制度

### 2024 年起新制

2024 年起，原有的快捷（Express）、普通學術（N(A)）、普通工藝（N(T)）分流取消，改為三個編班組別（Posting Group）：

| 編班組別 | 對應原分流 | 科目等級 | 說明 |
|----------|-----------|----------|------|
| Posting Group 3 | 快捷（Express） | G3（General 3） | 最高程度 |
| Posting Group 2 | 普通學術 N(A) | G2 | |
| Posting Group 1 | 普通工藝 N(T) | G1 | |

> **注意**：新制下，學生可依各科能力修讀不同等級科目（如數學修 G3、英語修 G2），不再固定於單一分流。

| 華僑中學（Hwa Chong Institution） | IP SAP（男校） | 6（M） |

> **注意**：(M) 表示高級華文（Higher Chinese）須達 Merit 等級。IP = Integrated Programme（直通車計劃，無須參加 O-Level）。

### 門檻說明

|------------|-------------|
| 4-6 | 頂尖 IP 中學 |

| 21-32 | 視個別情況 |

> **注意**：門檻分數每年依報名人數與考生成績而變動，以上僅為近年參考值。
