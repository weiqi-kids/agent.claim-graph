# GCSE 門檻數據

> General Certificate of Secondary Education（英國中等教育普通證書）

---

## 基本資訊

| 項目 | 內容 |
|------|------|
| 考試名稱 | GCSE（General Certificate of Secondary Education） |
| 主辦 / 監管 | Ofqual（監管）；AQA、Pearson Edexcel、OCR、WJEC、CCEA（考試局） |
| 考試對象 | 英格蘭 Year 11 學生（約 15-16 歲） |
| 考試科目數 | 通常 8-10 科（核心 + 選修） |
| 成績等級 | 9（最高）至 1（最低） |
| 必修科目 | 英語、數學、科學 |
| 考試時間 | 每年 5-6 月 |
| 資料年度 | 2023-2025 |

---

## 等級制說明

2017 年起，英格蘭 GCSE 由字母等級（A\*-G）改為數字等級（9-1）：

| 數字等級 | 約對應舊制 | 說明 |
|----------|-----------|------|
| 9 | 高於 A\* | 最頂尖表現 |
| 8 | A\*/A 之間 | |
| 7 | A | |
| 6 | B（高） | |
| 5 | B（低）/ C（高） | 「強及格」（Strong Pass） |
| 4 | C | 「標準及格」（Standard Pass） |
| 3 | D | |
| 2 | E/F | |
| 1 | F/G | |

> **注意**：等級 4 為升學與就業的「標準及格」門檻；等級 5 為政府績效指標採用的「強及格」門檻。

## 歷年趨勢

| 年度 | 等級 7+ | 等級 4+ | 備註 |
|------|---------|---------|------|
| 2025 | 21.8% | 67.1% | 恢復常態後第三年 |
| 2024 | 21.7% | 67.4% | |
| 2023 | 21.6% | 67.8% | 回歸疫前標準（2019 約 21.9% / 69.9%） |
| 2022 | ~26% | ~73% | 疫情期間膨脹 |
| 2019 | ~21% | ~67-70% | 疫前參考基準 |

**趨勢摘要**：2023 年起回歸疫前評分標準，等級 7+ 比例穩定在 21-22%，等級 4+ 穩定在 67-68%，已回到與 2019 年相近的水準。

> **注意**：2019 年與 2022 年的精確數據因統計口徑（全部考生 vs. 16 歲考生、全英國 vs. 英格蘭）而有差異，此處列出近似值。
