# 學測門檻數據

> 大學學科能力測驗 - 台灣大學入學考試

---

## 基本資訊

| 項目 | 內容 |
|------|------|
| 考試名稱 | 大學學科能力測驗（學測） |
| 主辦單位 | 大學入學考試中心 |
| 考試對象 | 高三學生 |
| 滿分 | 各科 15 級分 |
| 考試科目 | 國文、英文、數學A/B、社會、自然 |
| 資料年度 | 113 學年度 (2024) |

---

## 級分制說明

學測採用級分制，將考生成績分為 15 級：

| 級分 | 百分位約 | 說明 |
|------|----------|------|
| 15 級（滿級分） | 前 1% | 頂尖 |
| 14 級（頂標） | 前 12% | |
| 13 級（前標） | 前 25% | |
| 11 級（均標） | 前 50% | 中位數 |
| 8 級（後標） | 前 75% | |
| 5 級（底標） | 前 88% | |

## 數學 A vs 數學 B

| 項目 | 數學 A | 數學 B |
|------|--------|--------|
| 難度 | 較高 | 較低 |
| 適合 | 理工、商管 | 文法、設計、部分商管 |
| 範圍 | 含微積分、向量 | 不含微積分 |

## 數學技能對應

學測數學涵蓋高中數學課綱：

| 領域 | 對應內容 |
|------|----------|
| 代數 | 多項式、指數對數、數列級數 |
| 幾何 | 向量、解析幾何 |
| 機率統計 | 排列組合、機率、統計 |
| 微積分（數A） | 極限、微分、積分基礎 |
