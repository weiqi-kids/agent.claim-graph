# UKMT 門檻數據

> UK Mathematics Trust - 英國數學競賽

---

## 基本資訊

| 項目 | JMC | IMC | SMC |
|------|-----|-----|-----|
| 全名 | Junior Mathematical Challenge | Intermediate Mathematical Challenge | Senior Mathematical Challenge |
| 參賽對象 | Year 7-8 (11-13 歲) | Year 9-11 (14-16 歲) | Year 12-13 (16-18 歲) |
| 題數 | 25 題 | 25 題 | 25 題 |
| 滿分 | 135 分 | 135 分 | 125 分 |
| 時間 | 60 分鐘 | 60 分鐘 | 90 分鐘 |

## 計分方式

| 題號 | 正確 | 空白 | 錯誤 |
|------|------|------|------|
| 1-15 | +5 | 0 | 0 |
| 16-20 | +6 | 0 | -1 (SMC) / 0 (JMC/IMC) |
| 21-25 | +6 | 0 | -2 (SMC) / 0 (JMC/IMC) |

## 晉級路徑

### JMC 路線

```
JMC
  ↓ 金獎（前 6%）
JMO (Junior Mathematical Olympiad)
  ↓ 選拔
BMO1 (British Mathematical Olympiad Round 1)
```

### IMC 路線

```
IMC
  ↓ 金獎（前 6%）
Cayley/Hamilton/Maclaurin Olympiad（按年級）
  ↓ 選拔
BMO1
```

### SMC 路線

```
SMC
  ↓ 高分者
BMO1 (British Mathematical Olympiad Round 1)
  ↓ 高分者
BMO2 (British Mathematical Olympiad Round 2)
  ↓ 選拔
訓練營 → IMO 國家代表
```
