## 課程架構說明

Ontario Grade 4 數學課程分為五大領域（Strands）：
- **Number**（數）
- **Algebra**（代數）
- **Data**（資料）
- **Spatial Sense**（空間感）
- **Financial Literacy**（理財素養）

Grade 4 重點：
- 發展對乘除法關係的理解
- 識別並描述模式中的重複元素
- 使用量尺和平均數分析資料
- 發展對三維物體的空間感

## 學習內容標準

### Number（數）

| 學習內容 | 說明 |
|----------|------|
| 位值概念 | 理解至萬位的位值 |
| 乘除法關係 | 識別等值的乘除法式子（如 3×4 = 6×2） |
| 多位數運算 | 多位數的加減乘除 |
| 分數概念 | 理解分數、比較分數 |
| 小數概念 | 認識小數及其與分數的關係 |

### Algebra（代數）

| 學習內容 | 說明 |
|----------|------|
| 模式識別 | 識別並描述重複模式（如每週一發生的事件） |
| 數列規律 | 識別數列中的規律（如每次加 2） |
| 編程思維 | 撰寫程式碼執行重複運算（如從 0 開始重複加 3） |
| 數學建模 | 用數學模型分析真實情境（如規劃社區菜園） |

### Data（資料）

| 學習內容 | 說明 |
|----------|------|
| 資料收集 | 學習更多收集和組織資料的方法 |
| 圖表量尺 | 在圖表中使用量尺 |
| 平均數 | 使用平均數比較資料 |
| 資料分析 | 從圖表中提取和解釋資訊 |

### Spatial Sense（空間感）

| 學習內容 | 說明 |
|----------|------|
| 三維物體 | 認識和描述三維物體 |
| 物體分解 | 想像物體拆解或翻轉後的樣子 |
| 角度 | 認識和測量角度 |
| 面積與周長 | 計算長方形的面積和周長 |

### Financial Literacy（理財素養）

| 學習內容 | 說明 |
|----------|------|
| 金錢概念 | 理解消費和儲蓄的概念 |
| 預算基礎 | 簡單的預算規劃 |

## 與美國 Common Core Grade 4 的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 加拿大 Ontario Grade 4 | 美國 Common Core Grade 4 | 相似度 |
|------------------------|---------------------------|--------|
| 位值概念 | 4.NBT.A (Place value) | 高度相似 |
| 多位數運算 | 4.NBT.B (Operations) | 高度相似 |
| 分數概念 | 4.NF (Fractions) | 高度相似 |
| 面積與周長 | 4.MD.A.3 (Area and perimeter) | 高度相似 |
| 角度 | 4.MD.C (Angles) | 高度相似 |
| 編程思維 | 不在 Common Core | Ontario 特色 |
| 理財素養 | 不在 Common Core | Ontario 特色 |

**比較說明**：
- Ontario 和美國 Common Core 在核心數學內容上高度一致
- Ontario 從 Grade 1 就開始融入編程思維，這是獨特之處
- Ontario 將理財素養列為正式課綱領域
- Ontario 更強調數學建模和真實情境應用
