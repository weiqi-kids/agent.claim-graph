## 課程架構說明

日本中學數學課程分為四大領域：
- **A. 數與式**
- **B. 圖形**
- **C. 函數**
- **D. 資料的運用**

第二學年重點：
- 文字式的計算與式的變形
- 一次函數的理解與應用
- 平行線的性質與三角形的全等
- 機率的初步概念

### A. 數與式

| 學習內容 | 說明 |
|----------|------|
| 文字式的計算 | 單項式與多項式的加減乘除 |

| 學習內容 | 說明 |

| 一次函數的應用 | 運用一次函數解決實際問題 |

### D. 資料的運用

## 與美國 Common Core Grade 8 的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 日本中學二年級 | 美國 Common Core Grade 8 | 相似度 |
|----------------|---------------------------|--------|
| 文字式的計算 | 8.EE.A (Integer exponents) | 相似 |
| 聯立方程式 | 8.EE.C.8 (Systems of equations) | 高度相似 |
| 一次函數 | 8.F.A (Linear functions) | 高度相似 |
| 平行線的性質 | 8.G.A.5 (Parallel lines) | 高度相似 |
| 三角形全等 | 8.G.A.2 (Congruence) | 高度相似 |
| 機率 | 不在 Grade 8（在 Grade 7） | 日本較晚引入 |

**比較說明**：
- 日本和美國在代數（聯立方程式、一次函數）方面高度一致
- 日本更強調幾何證明的嚴謹性
- 美國在 Grade 8 已引入畢氏定理，日本通常在三年級教授
- 日本更重視機率概念的紮實理解

> **本比較僅呈現數據差異，不構成評估建議。**

- 日本更強調幾何證明的系統性教學
- 兩者都重視函數概念的建立
