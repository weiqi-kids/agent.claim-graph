## 課程架構說明

澳洲 Year 4 數學課程分為三大內容領域：
- **Number and Algebra**（數與代數）
- **Measurement and Space**（測量與空間）
- **Statistics and Probability**（統計與機率）

精熟能力（Proficiency Strands）：
- **Understanding**：建立數學概念之間的連結
- **Fluency**：發展計算的效率和準確性
- **Problem-solving**：運用數學解決真實情境問題
- **Reasoning**：發展邏輯思維和論證能力

## 學習內容標準

### Number and Algebra（數與代數）

| 學習內容 | 說明 |
|----------|------|
| 位值概念 | 認識萬位以上的數，理解位值系統 |
| 整數運算 | 多位數的加減法，使用心算和筆算策略 |
| 乘法事實 | 熟練乘法表至 10×10 |
| 乘除法 | 兩位數乘以一位數，簡單的除法 |
| 分數概念 | 認識分數、等值分數、分數比較 |
| 小數概念 | 認識小數至百分位，與分數的關係 |
| 數的模式 | 識別和延續數字模式 |

### Measurement and Space（測量與空間）

| 學習內容 | 說明 |
|----------|------|
| 長度、面積、體積 | 使用標準單位測量和比較 |
| 質量和容量 | 使用公斤、公升等單位 |
| 時間 | 讀取和計算時間（時、分） |
| 角度 | 認識角度類型（銳角、直角、鈍角） |
| 二維形狀 | 認識和分類二維形狀的性質 |
| 三維物體 | 認識三維物體及其展開圖 |
| 位置和方向 | 使用方位描述位置和路徑 |

### Statistics and Probability（統計與機率）

| 學習內容 | 說明 |
|----------|------|
| 資料收集 | 設計問題、收集和記錄資料 |
| 資料表示 | 使用表格、圖表呈現資料 |
| 資料分析 | 從資料中提取資訊，做出結論 |
| 機率語言 | 使用「可能」、「不可能」、「確定」等詞彙 |

## 與美國 Common Core Grade 4 的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 澳洲 Year 4 | 美國 Common Core Grade 4 | 相似度 |
|-------------|---------------------------|--------|

**比較說明**：
- 澳洲和美國在 Year 4 / Grade 4 的數學內容高度一致
- 澳洲更強調精熟能力的整合（理解、流暢、問題解決、推理）
- 美國在分數方面的要求略深
- 兩者都重視位值系統和基本運算的精熟
