## 課程架構說明

日本小學數學課程分為四大領域：
- **A. 數與計算**
- **B. 量與測量**
- **C. 圖形**
- **D. 數學關係**

第四學年重點：
- 掌握多位數的四則運算
- 理解分數和小數的概念
- 發展面積測量能力
- 初步理解角度概念

---

## 學習內容標準

### A. 數與計算

| 學習內容 | 說明 |
|----------|------|
| 大數的認識 | 認識億、兆等大數單位，理解十進位制 |
| 整數的乘除法 | 三位數乘以二位數、三位數除以二位數 |
| 計算的估算 | 使用概數進行估算 |
| 分數的理解 | 理解分數的意義，同分母分數的加減 |
| 小數的運算 | 小數的加減法，小數的意義擴展 |
| 四則運算的關係 | 理解加減乘除之間的關係 |

### B. 量與測量

| 學習內容 | 說明 |
|----------|------|
| 面積的概念 | 理解面積的意義，使用平方公分、平方公尺 |
| 面積公式 | 長方形、正方形的面積計算 |
| 角度的測量 | 使用量角器測量角度 |
| 角度的分類 | 認識銳角、直角、鈍角 |

### C. 圖形

| 學習內容 | 說明 |
|----------|------|
| 平行四邊形和梯形 | 認識平行四邊形、梯形的性質 |
| 菱形 | 認識菱形及其性質 |
| 垂直與平行 | 理解垂直線和平行線的概念 |
| 對角線 | 了解四邊形對角線的性質 |
| 立體圖形 | 認識立方體和長方體 |

### D. 數學關係

| 學習內容 | 說明 |
|----------|------|
| 變化規律 | 探索兩個量之間的變化關係 |
| 式的表示 | 用□或△表示未知數 |
| 資料整理 | 使用表格和圖表整理資料 |
| 折線圖 | 讀取和繪製折線圖 |

## 與美國 Common Core Grade 4 的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 日本小學四年級 | 美國 Common Core Grade 4 | 相似度 |
|----------------|---------------------------|--------|
| 大數的認識（億、兆） | 4.NBT (Place value to millions) | 相似 |
| 整數乘除法 | 4.NBT.B.4-6 (Multi-digit operations) | 高度相似 |
| 分數的理解 | 4.NF (Fractions) | 高度相似 |
| 小數的運算 | 4.NF.C (Decimal notation) | 高度相似 |
| 面積公式 | 4.MD.A.3 (Area and perimeter) | 高度相似 |
| 角度測量 | 4.MD.C.5-7 (Angle measurement) | 高度相似 |

**比較說明**：
- 日本和美國在四年級數學的核心內容高度一致
- 日本更強調估算和概數的應用
- 美國更強調分數的多元表徵
- 兩者都重視幾何圖形的性質探索
