## 課程架構說明

Ontario Grade 8 數學課程分為五大領域（Strands）：
- **Number**（數）
- **Algebra**（代數）
- **Data**（資料）
- **Spatial Sense**（空間感）
- **Financial Literacy**（理財素養）

Grade 8 重點：
- 引入整數（負數）概念
- 識別等速變化的模式
- 區分離散資料和連續資料
- 發展四邊形的性質和面積計算能力

## 學習內容標準

### Number（數）

| 學習內容 | 說明 |
|----------|------|
| 大數認識 | 認識百萬以上的數 |
| 整數 | 認識整數（如 -2, -1, 0, 1, 2） |
| 整除規則 | 學習 2, 3, 4, 5, 6, 8, 9, 10 的整除規則 |
| 分數除法 | 整數除以分數或帶分數（如 2½） |
| 混合運算 | 解決涉及整數、小數、分數的多步驟問題 |

### Algebra（代數）

| 學習內容 | 說明 |
|----------|------|
| 等速變化模式 | 識別以等速變化的模式（如每小時 100 公里） |
| 代數式 | 解涉及整數和小數十分位的代數式 |
| 代數方程式 | 解多項式代數方程式（如 2x + 3x = 5） |
| 編程優化 | 使用程式碼解決優化問題（如給定周長求最大面積） |

### Data（資料）

| 學習內容 | 說明 |
|----------|------|
| 資料類型 | 區分離散資料（如學生人數）和連續資料（如降水量） |
| 圖表類型 | 學習顯示時間變化的圖表（如折線圖） |
| 機率 | 學習不同描述機率的方式 |
| 資料分析 | 從資料中得出結論 |

### Spatial Sense（空間感）

| 學習內容 | 說明 |
|----------|------|
| 四邊形性質 | 學習不同四邊形的特徵和性質 |
| 四邊形面積 | 計算各種四邊形的面積 |
| 三維結構 | 建構三維結構，計算表面積 |
| 單位換算 | 公制單位之間的換算 |
| 角度性質 | 使用角度性質求未知角度 |
| 量角器 | 使用量角器測量角度 |

### Financial Literacy（理財素養）

| 學習內容 | 說明 |
|----------|------|
| 財務決策 | 理解財務決策的影響 |
| 利息概念 | 簡單利息的計算 |

## 與美國 Common Core Grade 8 的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 加拿大 Ontario Grade 8 | 美國 Common Core Grade 8 | 相似度 |
|------------------------|---------------------------|--------|
| 整數 | 不在 Grade 8（在 Grade 6-7） | 美國較早 |
| 代數方程式 | 8.EE.C (Linear equations) | 相似 |
| 等速變化模式 | 8.EE.B.5 (Proportional relationships) | 高度相似 |
| 四邊形性質 | 不在 Grade 8（在 Grade 7） | 美國較早 |
| 機率 | 不在 Grade 8（在 Grade 7） | 美國較早 |
| 編程優化 | 不在 Common Core | Ontario 特色 |
| 理財素養 | 不在 Common Core | Ontario 特色 |

**比較說明**：
- Ontario Grade 8 的數學內容較美國 Common Core Grade 8 稍淺
- Ontario 的代數發展較慢，Grade 8 仍在處理線性關係基礎
- 美國 Grade 8 已引入函數概念、畢氏定理等，Ontario 較晚引入
- Ontario 的獨特優勢在於編程思維和理財素養的整合

## 與台灣國中八年級的對應

> **本比較僅呈現數據差異，不構成評估建議。**

| 加拿大 Ontario Grade 8 | 台灣國中八年級 | 相似度 |
|------------------------|----------------|--------|
| 代數方程式 | 乘法公式、因式分解、一元二次方程式 | 台灣更深入 |
| 等速變化 | 一次函數 | 相似 |
| 幾何 | 畢氏定理、平行線性質 | 台灣更深入 |
| 機率 | 機率概念 | 相似 |

**比較說明**：
- 台灣國中八年級的數學內容明顯較 Ontario Grade 8 深入
- 台灣已教授一元二次方程式、畢氏定理等，Ontario 尚未涵蓋
- Ontario 更注重應用和真實情境，台灣更注重計算技巧
- 兩者的數學發展進度差異約 1-2 年
