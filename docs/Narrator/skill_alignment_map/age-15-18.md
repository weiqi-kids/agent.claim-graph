#### 資料來源
- [Common Core State Standards: High School Mathematics](https://www.thecorestandards.org/Math/)
- [AP Calculus AB](https://apstudents.collegeboard.org/courses/ap-calculus-ab)

#### 架構說明

Common Core 高中數學以 **6 大概念類別**（Conceptual Categories）組織，每類下分若干**領域**（Domains），每領域下分**群組**（Clusters）。Modeling 貫穿於其他 5 類中，不獨立列出標準。

#### 概念類別 1：Number and Quantity（數與量）

| 領域 | 群組 |
|------|------|
| The Real Number System (N-RN) | 將指數性質推廣到有理指數；使用有理數與無理數的性質 |
| Quantities (N-Q) | 量化推理與單位使用 |
| The Complex Number System (N-CN) | 複數的算術運算；在複數平面上表示複數及其運算；在多項式恆等式與方程中使用複數 |
| Vector Quantities and Matrices (N-VM) | 表示與建模向量量；向量運算；矩陣運算及矩陣應用 |

#### 概念類別 2：Algebra（代數）

| 領域 | 群組 |
|------|------|
| Seeing Structure in Expressions (A-SSE) | 解讀表達式的結構；以等價形式改寫表達式以解題 |
| Arithmetic with Polynomials & Rational Expressions (A-APR) | 多項式算術運算；理解零點與因式的關係；使用多項式恆等式解題；改寫有理式 |
| Creating Equations (A-CED) | 建立描述數量或關係的方程式 |
| Reasoning with Equations & Inequalities (A-REI) | 理解解方程是推理過程；解一元方程與不等式；解方程組；以圖形表示並求解方程與不等式 |

#### 概念類別 3：Functions（函數）

| 領域 | 群組 |
|------|------|
| Interpreting Functions (F-IF) | 理解函數概念與函數記號；以不同表徵分析函數 |
| Building Functions (F-BF) | 從已知函數建構新函數 |
| Linear, Quadratic & Exponential Models (F-LE) | 建構並比較線性、二次與指數模型 |
| Trigonometric Functions (F-TF) | 以單位圓推廣三角函數定義域；以三角函數建模週期現象；證明並應用三角恆等式 |

#### 概念類別 4：Geometry（幾何）

| 領域 | 群組 |
|------|------|
| Congruence (G-CO) | 用幾何變換理解全等；證明幾何定理；以作圖驗證 |
| Similarity, Right Triangles & Trigonometry (G-SRT) | 以相似變換理解相似；用三角比解直角三角形；應用三角學 |
| Circles (G-C) | 理解並應用圓的性質（內接角、弦、切線） |
| Expressing Geometric Properties with Equations (G-GPE) | 以方程式表達幾何性質（含圓錐曲線推導） |
| Geometric Measurement & Dimension (G-GMD) | 以非正式論證解釋體積公式 |
| Modeling with Geometry (G-MG) | 應用幾何概念建模 |

#### 概念類別 5：Statistics & Probability（統計與機率）

| 領域 | 群組 |
|------|------|
| Interpreting Categorical & Quantitative Data (S-ID) | 摘要、表示與解讀單變量、雙變量資料 |
| Making Inferences & Justifying Conclusions (S-IC) | 理解並評估抽樣調查、實驗與觀察研究 |
| Conditional Probability & the Rules of Probability (S-CP) | 理解獨立性與條件機率 |
| Using Probability to Make Decisions (S-MD) | 計算隨機變量期望值；以機率輔助決策 |

#### 概念類別 6：Modeling（建模）

Modeling 不獨立列標準，而是貫穿於上述五大類中，凡標有星號 (*) 的標準均與建模相關。

#### AP Calculus AB（選修，大學先修）

| Unit 4 | Contextual Applications of Differentiation — 相關變率、線性近似 | 10-15% |
| Unit 5 | Analytical Applications of Differentiation — 均值定理、極值、凹凸性 | 15-18% |
| Unit 6 | Integration and Accumulation of Change — 黎曼和、定積分、微積分基本定理 | 17-20% |
| Unit 7 | Differential Equations — 分離變數法、指數增長/衰退模型 | 6-12% |
| Unit 8 | Applications of Integration — 面積、體積、累積量 | 10-15% |

#### AP Calculus BC（含 AB 全部 + 以下額外內容）

#### 架構說明

108 課綱高中數學必修 16 學分 + 選修 8 學分，共 24 學分。高一共同課程，高二起分為**數學 A**（自然組/理工）和**數學 B**（社會組/文法商），高三進入選修**數學甲**（理工深化）或**數學乙**（商管應用）。課綱採「螺旋性架構」，同一主題分階段深化。

#### 高一（共同必修 8 學分）— 數學 I、II

| 冊 | 主題 | 內容 |
|----|------|------|
| 數學 I（上） | 數與式 | 實數、乘法公式（含三次）、根式運算 |
| | 指數律 | 指數律、常用對數 |
| | 多項式函數 | 多項式除法、餘式定理、三次函數圖形 |
| | 直線與圓 | 斜率、方程式、圓的方程式 |
| 數學 II（下） | 數列與級數 | 等差數列、等比數列、級數求和 |

| | 數據分析 | 統計量、百分位數、期望值 |
| | 三角比 | 直角三角形三角比、廣義角三角比、正弦/餘弦定理 |

#### 高二（分流必修各 4 學分）— 數學 III、IV

**數學 A（自然組方向）**

| 冊 | 主題 | 內容 |
|----|------|------|
| 數學 III-A（上） | 三角函數 | 弧度制、三角函數圖形、和差角公式、正餘弦疊合 |

| 數學 IV-A（下） | 平面向量 | 向量運算、內積、面積 |
| | 空間向量 | 空間坐標、空間向量、平面與直線方程式 |
| | 條件機率 | 條件機率、貝氏定理 |

**數學 B（社會組方向）**

| 冊 | 主題 | 內容 |
|----|------|------|
| 數學 III-B（上） | 三角函數 | 弧度制、三角函數圖形（較簡化） |

#### 高二數學 A 與數學 B 的主要差異

| 主題 | 數學 A | 數學 B |
|------|--------|--------|
| 三角函數 | 含和差角公式、疊合 | 較簡化，不含疊合 |

#### H2 Mathematics（9758）— 純數 + 統計

**Section A: Pure Mathematics（考試佔 40 分）**

| 主題 | 內容 |
|------|------|
| Functions and Graphs | 函數定義域/值域、複合函數、反函數、圖形變換（平移/伸縮/反射）、有理函數圖形 |

**Section B: Probability and Statistics（考試佔 60 分）**

| 主題 | 內容 |
|------|------|
| Probability | 加法/乘法法則、條件機率、獨立事件、全機率公式 |

| Sampling and Estimation | 抽樣分布、中央極限定理、信賴區間 |
| Hypothesis Testing | 假設檢定（均值）、顯著水準、型一/型二錯誤 |
| Correlation and Regression | 散佈圖、積差相關係數、最小平方迴歸線 |

<summary>🇬🇧 英國 GCSE + A-Level Mathematics（點擊展開）</summary>

#### 資料來源

#### GCSE Mathematics（Year 10-11，約 15-16 歲，必修）

GCSE 分 Foundation（Grades 1-5）與 Higher（Grades 4-9）兩層級，3 張考卷（1 張不可用計算機 + 2 張可用計算機）。

| 主題 | 佔比（Foundation / Higher） | 內容 |
|------|---------------------------|------|
| Number | 25% / 15% | 整數、分數、小數、百分比、因數與倍數、標準記數法、近似值 |
| Algebra | 20% / 30% | 表達式、方程式、不等式、數列、函數圖形、二次方程 |
| Ratio, Proportion & Rates of Change | 25% / 20% | 比、正反比、百分比變化、增長/衰退、單位換算 |
| Geometry & Measures | 15% / 20% | 角度、面積、體積、變換、畢氏定理、三角學、向量 |
| Probability | 約 7.5% / 7.5% | 機率基礎、樣本空間、樹狀圖、Venn 圖 |
| Statistics | 約 7.5% / 7.5% | 資料蒐集、圖表、平均數/中位數/眾數、散佈圖 |

#### A-Level Mathematics（Year 12-13，約 16-18 歲，選修）

3 張考卷（各 2 小時、100 分），佔 A-Level 各 33⅓%。

**Paper 1: Pure Mathematics（Sections A-I）**

| 章節 | 主題 | 內容 |
|------|------|------|
| A | Proof | 演繹推理、反證法、反例 |
| B | Algebra and Functions | 指數法則、函數圖形變換、因式分解、部分分式、模數函數 |
| C | Coordinate Geometry | 直線、圓、參數方程 |
| D | Sequences and Series | 等差/等比數列、二項展開式（含有理指數）、Sigma 記號 |
