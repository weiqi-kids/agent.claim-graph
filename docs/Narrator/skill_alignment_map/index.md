# 年級技能標準

依據美國 Common Core 課程標準，列出各年級學生應掌握的技能。

> **語文定義**：對當地主流母語的掌握能力（美國為英語）

## 可用報告

### 數學 (Mathematics)

| 年級 | 對應台灣 | 技能數 |
|------|----------|--------|
| [國小四年級數學](Grade-4-Math) | 國小四年級 | 28 項 |
| [國中二年級數學](Grade-8-Math) | 國中二年級 | 28 項 |

### 語文 (English Language Arts)

| 年級 | 對應台灣 | 技能數 |
|------|----------|--------|
| [國小四年級語文](Grade-4-ELA) | 國小四年級 | 45 項 |
| [國中二年級語文](Grade-8-ELA) | 國中二年級 | 42 項 |

> **說明**：美國 Grade 4 約等於台灣國小四年級，Grade 8 約等於國中二年級。

## 資料來源

所有資料來自 [Common Core State Standards](https://www.thecorestandards.org/)，為美國大多數州採用的課程標準。
