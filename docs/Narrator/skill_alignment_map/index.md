## 課程特色比較

| 國家 | 課程特色 |
|------|----------|
| **美國** | 強調概念理解與多元表徵，重視數學推理與問題解決 |
| **台灣** | 強調計算熟練度與代數技巧，較早教授二次方程式 |
| **新加坡** | 採用 CPA 教學法（具體→圖像→抽象），強調 Bar Model 解題 |
| **英國** | 強調流暢性、推理、問題解決三大核心，重視幾何變換 |
| **日本** | 重視估算應用與幾何證明的嚴謹性 |
| **加拿大** | 融入編程思維與理財素養（Ontario 特色） |
| **澳洲** | 四項精熟能力（理解、流暢、問題解決、推理） |
