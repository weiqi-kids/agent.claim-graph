## 課程架構說明

英國國家課程將小學數學分為 Key Stage 1 (Years 1-2) 和 Key Stage 2 (Years 3-6)。

Year 5 屬於 Key Stage 2 的第三年，課程重點在於：
- 鞏固大數的位值概念（至百萬）
- 擴展分數和小數的理解至千分位
- 發展複雜的問題解決能力

英國課程的三大核心目標：
1. **Fluency** (流暢性) - 概念理解與快速計算
2. **Reasoning** (推理) - 數學論證與關係探究
3. **Problem Solving** (問題解決) - 應用於各種情境

## 學習內容標準

### 一、Number and Place Value (數與位值)

| 學習內容 | 說明 |
|----------|------|
| Read, write and order numbers to 1,000,000 | 讀寫和排序至百萬的數字，並確定每一位數的值 |
| Count forwards/backwards in powers of 10 | 對任意數字進行十的次方倍數的順數或倒數 |
| Round numbers to nearest 10, 100, 1000, 10,000, 100,000 | 將數字四捨五入至最接近的十、百、千、萬、十萬 |
| Interpret negative numbers in context | 在情境中解釋負數（如溫度、海拔） |
| Determine value of digits in numbers | 確定數字中每一位的值 |

### 二、Addition and Subtraction (加法與減法)

| 學習內容 | 說明 |
|----------|------|
| Add and subtract whole numbers (>1,000,000) | 加減超過百萬的整數 |
| Use rounding to check answers | 使用四捨五入檢查答案的合理性 |
| Solve multi-step problems | 解決多步驟加減問題 |
| Mental calculations with large numbers | 大數的心算策略 |

### 三、Multiplication and Division (乘法與除法)

| 學習內容 | 說明 |
|----------|------|
| Multiplication facts to 12 × 12 | 熟記乘法表至 12×12 |
| Multiply numbers up to 4 digits by 1 or 2-digit numbers | 四位數乘以一位或兩位數 |
| Divide numbers up to 4 digits by 1-digit numbers | 四位數除以一位數 |
| Multiply and divide by 10, 100, 1000 | 乘除以 10、100、1000 |
| Recognize square and cube numbers | 認識平方數和立方數 |
| Prime numbers up to 100 | 認識至 100 的質數 |
| Factors and multiples | 因數與倍數 |

### 四、Fractions (分數)

| 學習內容 | 說明 |
|----------|------|
| Compare and order fractions | 比較與排序分數（分母不同） |
| Equivalent fractions | 認識等值分數 |
| Convert mixed numbers and improper fractions | 轉換帶分數與假分數 |
| Add and subtract fractions | 加減分數（分母相同或相關） |
| Multiply fractions by whole numbers | 分數乘以整數 |
| Write fractions as decimals | 將分數寫成小數（分母為 10 或 100） |

### 五、Decimals and Percentages (小數與百分數)

| 學習內容 | 說明 |
|----------|------|
| Read, write and order decimals (3 decimal places) | 讀寫和排序至千分位的小數 |
| Recognize equivalence: tenths, hundredths, thousandths | 認識十分位、百分位、千分位的等值關係 |
| Round decimals | 將小數四捨五入至整數或一位小數 |
| Compare decimals | 比較至三位小數 |
| Understand percentages | 理解百分數與「百分之一」的關係 |
| Write percentages as fractions/decimals | 將百分數寫成分數或小數 |

### 六、Measurement (測量)

| 學習內容 | 說明 |
|----------|------|
| Convert between metric units | 轉換公制單位（公里/公尺/公分/公釐、公斤/公克等） |
| Understand approximate equivalences (metric-imperial) | 理解公制與英制的大約等值 |
| Measure and calculate perimeter | 測量與計算多邊形的周長 |
| Calculate area of rectangles | 計算長方形的面積（平方公分/公尺） |
| Estimate volume and capacity | 估計體積與容量 |

### 七、Geometry - Properties of Shapes (幾何 - 形狀性質)

| 學習內容 | 說明 |

| Angles in degrees | 認識角度（銳角、鈍角、直角） |
| Draw angles and measure with protractor | 使用量角器繪製和測量角度 |
| Calculate angles on a straight line | 計算直線上的角度（和為 180°） |
| Calculate angles around a point | 計算一點周圍的角度（和為 360°） |
| Distinguish regular and irregular polygons | 區分正多邊形與不規則多邊形 |

### 八、Geometry - Position and Direction (幾何 - 位置與方向)

| 學習內容 | 說明 |
|----------|------|
| Identify, describe and represent positions | 在坐標系的第一象限識別和描述位置 |
| Plot specified points | 繪製指定的點 |
| Complete shapes with given coordinates | 用給定坐標完成形狀 |
| Translation | 平移變換 |
| Reflection | 鏡射變換 |

### 九、Statistics (統計)

| 學習內容 | 說明 |
|----------|------|
| Solve comparison problems | 使用資訊解決比較、總和、差異問題 |
| Complete and interpret information in tables | 完成並解釋表格中的資訊 |
| Line graphs | 繪製與解讀折線圖 |
| Timetables | 讀懂時刻表 |

## 與美國 Common Core Grade 4 的對應

英國 Year 5 學生年齡約為 9-10 歲，對應美國 Grade 4（同年齡）。

| 英國 Year 5 | 美國 Common Core Grade 4 | 相似度 |
|-------------|---------------------------|--------|
| Numbers to 1,000,000 | Numbers to 1,000,000 (4.NBT.A.2) | 高度相似 |
| Multi-digit operations | 4.NBT.B (Operations) | 高度相似 |
| Fractions | 4.NF (Fractions) | 相似，英國更深入 |
| Decimals to thousandths | 4.NF.C (Decimals to hundredths) | 英國更深入 |
| Angles | 4.MD.C (Angles) | 高度相似 |
| Area and perimeter | 4.MD.A.3 | 高度相似 |
| Percentages | 不在 Grade 4 | 英國較早引入 |

**比較說明**：
- 英國 Year 5 在分數和小數方面更為深入（至千分位）
- 英國較早引入百分數概念
- 英國強調坐標系和變換（平移、鏡射）
- 美國更強調分數的概念理解和應用
