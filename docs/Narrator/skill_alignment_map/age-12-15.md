# 12-15 歲學習地圖

涵蓋 7 國數學課程標準、2 項數學競賽。

> **資料可用性**：數學方面，7 國均有約 Grade 8 等效資料（13-14 歲）；Grade 7 等效資料目前僅美國有詳細標準。語文方面，已有美國、新加坡、英國、加拿大、澳洲 5 國資料（2026-03-05 更新）；台灣國語文、日本國語待補充。科學方面，目前僅有美國資料，其他國家待補充。

### 課程特色比較

| 國家 | 課程特色 |
|------|----------|
| 🇺🇸 **美國** | 強調函數概念、變換幾何、統計推論，以概念理解為導向 |
| 🇹🇼 **台灣** | 代數深度最高（已教二次方程式），強調乘法公式與因式分解技巧 |
| 🇸🇬 **新加坡** | 代數與幾何並重，已引入三角學，CPA 教學法持續應用 |
| 🇬🇧 **英國** | 均衡發展，機率教學較深（含機率樹），強調變換幾何 |
| 🇯🇵 **日本** | 重視幾何證明的嚴謹性，函數與方程式並進 |
| 🇨🇦 **加拿大** | 內容較淺但強調應用，融入編程與理財素養 |
| 🇦🇺 **澳洲** | 與美國高度一致，均含畢氏定理、指數、變換 |

#### 架構總覽

英國 KS3 英語課程為法定課程 (statutory)，分為 4 大領域。總體目標是培養學生高水準的語言和讀寫能力，並透過廣泛閱讀培養對文學的熱愛。

#### Reading 閱讀

| 要求類別 | 法定內容 |
|----------|----------|
| 閱讀範圍 | 閱讀各種類型的完整書籍，涵蓋廣泛的體裁、歷史時期、形式和作者 |
| Shakespeare | 至少研讀 **2 部** Shakespeare 劇作 |
| 文學時期 | 高品質英語文學作品，含 **1914 年以前** 及 **當代** 作品（散文、詩歌、戲劇） |
| 世界文學 | Seminal world literature（具里程碑意義的世界文學） |
| 詩歌 | 1789 年以來的詩歌，含代表性的 **浪漫主義詩歌 (Romantic poetry)** |
| 非文學類 | 論說文、新聞、傳記、自傳；參考資料及教科書文本 |
| 閱讀理解 | 精讀與泛讀；為愉悅與為獲取資訊而閱讀 |
| 批判閱讀 | 了解作者如何透過語言、結構和形式塑造意義 |
| 推論技能 | 做出推論並引用文本證據支持觀點 |
| 跨文本比較 | 比較兩個或以上文本的主題、風格和語言 |

#### Writing 寫作

| 要求類別 | 法定內容 |
|----------|----------|
| 寫作目的 | 為不同目的和受眾寫作，涵蓋正式與非正式語域 |
| 寫作類型 | 正式學術論文、議論文、說明文、描述文、敘事文 |
| 計劃與修訂 | 規劃、起草、編輯、校對的完整寫作歷程 |
| 長篇寫作 | 培養持久寫作的耐力和技巧 |
| 創意寫作 | 富想像力的寫作，展現對敘事和詩歌的掌握 |
| 筆記與摘要 | 有效記錄和整理資訊 |
| 正確性 | 準確的拼寫、標點和語法 |

#### Grammar and Vocabulary 語法與詞彙

| 要求類別 | 法定內容 |
|----------|----------|
| 語句結構 | 短語和子句的結構；透過 coordination 和 subordination 構建複雜句 |
| 段落結構 | 段落結構以及如何為不同文體建構段落 |
| Standard English | 控制口語和書面語中 Standard English 的使用 |
| 語域差異 | 理解正式/非正式語域之間的差異，以及 Standard English 與其他英語變體的差異 |
| 語言術語 | 學習正確的語法術語並整合於教學中 |
| 修辭手法 | 理解句子結構如何產生意義與效果 |
| 詞彙擴展 | 從閱讀和聆聽中汲取新詞彙和語法結構 |
| 語言學概念 | 理解語言隨時間演變以及語言變異的概念 |

#### Spoken English 口語英語

| 要求類別 | 法定內容 |
|----------|----------|
| 正式討論 | 參與正式辯論和結構化討論 |
| 總結與建構 | 總結和/或建基於他人的發言 |
| 即興發言 | 在正式和非正式情境中自信使用 Standard English |
| 呈現 | 使用說明、描述和敘述等模式進行簡短報告 |
| 角色扮演 | 即興表演、排練和演出戲劇及劇本 |
| 聆聽與回應 | 積極聆聽並對他人的觀點作出回應 |

|------|------|

#### Grade 7-8 與小學階段的關鍵差異

| 代碼 | 內容 |
|------|------|
| 1-Ⅳ-1 | 以同理心，聆聽各項發言，並加以記錄、歸納。 |

| 代碼 | 內容 |
