## 課程架構說明

澳洲 Year 8 數學課程分為三大內容領域：
- **Number and Algebra**（數與代數）
- **Measurement and Space**（測量與空間）
- **Statistics and Probability**（統計與機率）

Year 8 的精熟能力重點：
- **Understanding**：描述指數和循環小數的模式
- **Fluency**：準確計算簡單小數、指數和整數
- **Problem-solving**：建模涉及比率、利潤和損失的實際情境
- **Reasoning**：進行數學論證和驗證

## 學習內容標準

### Number and Algebra（數與代數）

| 學習內容 | 說明 |
|----------|------|
| 指數法則 | 認識和應用指數法則 |
| 有理數與無理數 | 描述有理數和無理數 |
| 比率和比例 | 解決涉及比率的日常問題 |
| 百分比 | 計算百分比變化、利潤和損失 |
| 代數式 | 展開和因式分解簡單代數式 |
| 線性方程式 | 解一元一次方程式 |
| 線性圖形 | 繪製和分析線性函數圖形 |
| 坐標幾何 | 在坐標平面上工作 |

### Measurement and Space（測量與空間）

| 學習內容 | 說明 |
|----------|------|
| 面積和周長 | 計算複合形狀的面積和周長 |
| 圓的性質 | 圓周長和面積的計算 |
| 體積和表面積 | 計算柱體的體積和表面積 |
| 畢氏定理 | 理解和應用畢氏定理 |
| 全等和相似 | 認識全等和相似形狀 |
| 變換 | 平移、旋轉、反射、放大 |

### Statistics and Probability（統計與機率）

| 學習內容 | 說明 |
|----------|------|
| 資料分析 | 使用統計量描述資料集 |
| 資料呈現 | 選擇適當的圖表類型 |
| 雙變量資料 | 分析兩個變量之間的關係 |
| 機率計算 | 計算複合事件的機率 |

## Year 8 成就標準

根據 ACARA，Year 8 學生應能：
- 解決涉及比率、比例和百分比的日常問題
- 認識指數法則
- 描述有理數和無理數
- 建模涉及利潤和損失的實際情境
- 應用畢氏定理解決問題

| 澳洲 Year 8 | 美國 Common Core Grade 8 | 相似度 |

- 澳洲和美國在 Year 8 / Grade 8 的數學內容高度一致
- 兩者都涵蓋指數、有理數/無理數、線性函數、畢氏定理
- 美國在函數概念上略深入
- 澳洲的機率教學在 Year 8 仍持續發展

- 兩者在畢氏定理和線性函數方面一致
- 澳洲更強調統計和機率的應用
