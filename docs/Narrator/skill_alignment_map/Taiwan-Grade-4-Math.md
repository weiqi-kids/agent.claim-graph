## 課程架構說明

台灣十二年國教數學領域將學習內容分為以下主題：
- **N**（數與量）
- **S**（空間與形狀）
- **R**（關係）- 國小階段專用
- **D**（資料與不確定性）

第二學習階段（3-4年級）的重點：
- 數方面：確實掌握自然數的四則與混合運算，培養流暢的數字感，並初步學習分數與小數
- 量方面：以長度為基礎，學習量的常用單位及其計算
- 幾何方面：發展以角、邊要素認識幾何圖形的能力，並能以操作認識幾何圖形的性質

## 學習內容標準

### 一、數與量（Number and Quantity）

| 學習內容 | 說明 |
|----------|------|
| N-4-1 一億以內的數 | 認識一億以內的數，並能讀、寫、比較大小 |
| N-4-2 整數四則運算 | 能熟練整數加減乘除的四則運算及其應用 |
| N-4-3 四則混合計算 | 理解運算順序規則，能進行四則混合計算 |
| N-4-4 概數與估算 | 能用四捨五入法對大數取概數，並應用於估算 |
| N-4-5 分數的認識 | 認識真分數、假分數、帶分數，能進行分數的大小比較 |
| N-4-6 等值分數 | 理解等值分數的概念，能找出等值分數 |
| N-4-7 分數的加減 | 能計算同分母分數的加減運算 |
| N-4-8 小數的認識 | 認識二位小數，能進行小數的讀寫與大小比較 |
| N-4-9 小數的加減 | 能計算二位小數的加減運算 |
| N-4-10 整數倍數與因數 | 理解倍數和因數的概念 |

### 二、空間與形狀（Space and Shape）

| 學習內容 | 說明 |
|----------|------|
| S-4-1 角度的認識 | 認識量角器，能測量和繪製角度 |
| S-4-2 角度的計算 | 能進行角度的加減計算 |
| S-4-3 垂直與平行 | 認識垂直和平行線的概念與性質 |
| S-4-4 平面圖形 | 認識四邊形（含平行四邊形、梯形）的性質 |
| S-4-5 三角形分類 | 能依邊長和角度對三角形進行分類 |
| S-4-6 周長與面積 | 理解周長與面積的概念，能計算正方形、長方形的周長和面積 |
| S-4-7 平方公分 | 認識面積單位「平方公分」，能進行面積的測量與計算 |
| S-4-8 立體圖形 | 認識正方體、長方體的構成要素 |

### 三、關係（Relationships）

| 學習內容 | 說明 |
|----------|------|
| R-4-1 數的規律 | 能找出數字序列的規律，並延續數列 |
| R-4-2 圖形規律 | 能找出圖形排列的規律 |
| R-4-3 兩量關係 | 能理解兩個量之間的對應關係 |

### 四、資料與不確定性（Data and Uncertainty）

| 學習內容 | 說明 |
|----------|------|
| D-4-1 長條圖與折線圖 | 能報讀與製作長條圖和折線圖 |
| D-4-2 資料整理 | 能將資料整理成表格或圖表 |

## 與美國 Common Core Grade 4 的對應

| 台灣課綱 | 美國 Common Core | 相似度 |
|----------|------------------|--------|
| 整數四則運算 | 4.NBT.B.4-6 (Multi-digit operations) | 高度相似 |
| 分數的認識與運算 | 4.NF.A-C (Fractions) | 高度相似 |
| 小數的認識 | 4.NF.C.5-7 (Decimal notation) | 高度相似 |
| 角度測量 | 4.MD.C.5-7 (Angle measurement) | 高度相似 |
| 面積與周長 | 4.MD.A.3 (Area and perimeter) | 高度相似 |
| 因數與倍數 | 4.OA.B.4 (Factors and multiples) | 相似 |

**比較說明**：台灣四年級課程與美國 Common Core Grade 4 在核心概念上高度一致，主要差異在於：
- 台灣課綱更強調計算的熟練度
- 美國課綱更強調概念理解和多元表徵
- 兩者都重視分數、小數、幾何和測量的基礎
