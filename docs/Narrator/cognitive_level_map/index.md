# 考試題型分析

各國考試的題目，分別測驗哪種程度的思考能力？

## 題目類型說明

| 類型 | 在考什麼 | 題目特徵 |
|------|----------|----------|
| **記憶** | 知不知道 | 問定義、公式、名詞 |
| **理解** | 懂不懂 | 問「為什麼」「這代表什麼」 |
| **應用** | 會不會用 | 要求計算、解題 |
| **分析** | 能不能拆解 | 要求比較、推論、找關係 |
| **評鑑** | 能不能判斷 | 要求評估對錯、驗證 |
| **創造** | 能不能發明 | 要求設計新方法 |

**層次越高，越需要「思考」而非「記憶」。**
