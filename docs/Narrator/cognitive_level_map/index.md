# 認知層次分析

分析各國考試題目的 Bloom 認知層次分布，了解題目測驗的能力層次。

## Bloom 認知層次

| 層次 | 說明 | 關鍵動詞 |
|------|------|----------|
| 記憶 | 回憶事實、術語 | 列出、定義、辨認 |
| 理解 | 解釋意義 | 解釋、摘要、舉例 |
| 應用 | 用於新情境 | 計算、解決、執行 |
| 分析 | 拆解結構 | 比較、對照、推論 |
| 評鑑 | 判斷、驗證 | 評估、辯護、批評 |
| 創造 | 產生新想法 | 設計、發明、規劃 |
