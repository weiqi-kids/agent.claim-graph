## 怎麼使用這些資料？

1. **確認年齡帶**：孩子幾歲？找到對應的年齡帶頁面
2. **查看課綱**：該年齡帶各國教什麼？孩子掌握到哪裡？
3. **了解門檻**：如果有考試目標，需要什麼分數？
4. **跨國比較**：不同國家的教學重點有何不同？

這些資料讓你用**具體數字**評估孩子的程度，而不是憑感覺。
