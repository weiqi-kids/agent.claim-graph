# load_summary Mode

## Mode 可驗證性：Factual

> **Phase 4** — 此 Mode 在 Phase 1-3 為 `.disabled` 狀態

---

## Mode 定義表

| 項目 | 內容 |
|------|------|
| **Mode name** | load_summary（負荷摘要） |
| **Verifiability** | Factual |
| **Purpose and audience** | 彙整使用者輸入的身心指標數據 |
| **Source layers** | L7 wellbeing_signals |
| **Required disclaimer** | 見下方警語 |
| **Reviewer persona** | 自動化邊界審核員、使用者誤導風險審核員 |

---

## 必要警語

> **重要提示**
>
> 本報告僅彙整您輸入的身心指標數據，呈現趨勢變化。
>
> - 這不是心理健康評估或診斷
> - 系統不會提供任何心理健康建議
> - 如有身心健康疑慮，請諮詢專業人員（醫師、心理師、學校輔導老師）
>
> 專業諮詢資源：
> - 衛福部安心專線：1925
> - 張老師專線：1980

---

## 資料來源定義

從 `docs/Extractor/wellbeing_signals/` 讀取使用者輸入的週報資料。

---

## 禁止行為

- 不可提供心理健康建議
- 不可詮釋數據意義（如「壓力偏高」）
- 不可建議改變學習計畫
- 不可預測未來趨勢
